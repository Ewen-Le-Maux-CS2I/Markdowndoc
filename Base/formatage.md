# Format de texte

Markdown offre plusieurs options pour formater votre texte.

## Syntaxe

```markdown
**bold text**
*italic text*
***bold and italic***
~~strikethrough~~
`inline code`
```

## Rendu

**bold text**

*italic text*

***bold and italic***

~~strikethrough~~

`inline code`

## Exemples d'utilisation

- **Gras** : pour mettre en évidence des mots importants
- *Italique* : pour les citations ou l'emphase légère
- ***Gras et italique*** : pour une emphase maximale
- ~~Barré~~ : pour indiquer du texte supprimé ou obsolète
- `Code inline` : pour les noms de variables, fonctions ou commandes
