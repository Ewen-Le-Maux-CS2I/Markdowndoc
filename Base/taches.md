# Listes de tâches

Les listes de tâches (task lists) permettent de créer des listes d'éléments avec des cases à cocher.

## Syntaxe

```markdown
- [x] Completed task
- [ ] Incomplete task
- [ ] Another task
```

## Rendu

- [x] Completed task
- [ ] Incomplete task
- [ ] Another task

## Exemples pratiques

### Gestion de projet

- [x] Définir les objectifs
- [x] Créer le cahier des charges
- [ ] Développer la fonctionnalité A
- [ ] Développer la fonctionnalité B
- [ ] Tests unitaires
- [ ] Tests d'intégration
- [ ] Déploiement

### Liste de courses

- [x] Pain
- [x] Lait
- [ ] Œufs
- [ ] Fromage
- [ ] Fruits

### Apprentissage

- [x] Apprendre la syntaxe de base Markdown
- [x] Maîtriser les titres
- [ ] Créer des tableaux complexes
- [ ] Utiliser les extensions avancées

## Listes de tâches imbriquées

Vous pouvez imbriquer des sous-tâches :

- [ ] Projet principal
  - [x] Sous-tâche 1
  - [x] Sous-tâche 2
  - [ ] Sous-tâche 3
- [ ] Autre projet
  - [ ] Tâche A
  - [ ] Tâche B

## Bonnes pratiques

- Utilisez les listes de tâches pour le suivi de progression
- Mettez à jour régulièrement l'état des tâches
- Décomposez les grandes tâches en sous-tâches
- Idéal pour les README, issues GitHub, et documentation de projet
