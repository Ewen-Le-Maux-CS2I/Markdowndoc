# Liens et Images

## Liens

### Syntaxe

```markdown
[Link text](https://example.com)
[Link with title](https://example.com "Hover title")
```

### Exemples

[Visitez GitHub](https://github.com)

[GitHub](https://github.com "Plateforme de développement")

## Images

### Syntaxe

```markdown
![Alt text](image.jpg)
![Image with title](image.jpg "Image title")
```

### Bonnes pratiques

- Toujours fournir un texte alternatif (alt text) pour l'accessibilité
- Utiliser des chemins relatifs pour les images locales
- Optimiser la taille des images pour le web
- Ajouter un titre pour plus de contexte (optionnel)
