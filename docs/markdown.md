# Le langage Markdown

## Les titres
```
# H1 Header
## H2 Header
### H3 Header
#### H4 Header
##### H5 Header
###### H6 Header
```

## Format de text
```
**bold text**
*italic text*
***bold and italic***
~~strikethrough~~
`inline code`
```

## Liens et Images
```
[Link text](https://example.com)
[Link with title](https://example.com "Hover title")
![Alt text](image.jpg)
![Image with title](image.jpg "Image title")
```

## Listes
```
Unordered:
- Item 1
- Item 2
  - Nested item

Ordered:
1. First item
2. Second item
3. Third item
```

## Blocs ne notes
```
> This is a blockquote
> Multiple lines
>> Nested quote
```

## Blocs de code
````
```javascript
function hello() {
  console.log("Hello, world!");
}
```
````

## Tableau
```
| Header 1 | Header 2 | Header 3 |
|----------|----------|----------|
| Row 1    | Data     | Data     |
| Row 2    | Data     | Data     |
```

## Regles Horizontale
```
---
or
***
or
___
```

## List de tâches
```
- [x] Completed task
- [ ] Incomplete task
- [ ] Another task
```

## Charactère d'échappement
```
Use backslash to escape: \* \_ \# \`
```

## Saut de ligne
```
End a line with two spaces  
to create a line break.

Or use a blank line for a new paragraph.
```
