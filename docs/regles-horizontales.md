# Règles horizontales

Les règles horizontales (ou séparateurs) permettent de diviser le contenu en sections distinctes.

## Syntaxe

Il existe trois façons de créer une règle horizontale :

```markdown
---
```

ou

```markdown
***
```

ou

```markdown
___
```

## Rendu

Toutes ces syntaxes produisent le même résultat :

---

## Utilisation

Les règles horizontales sont utiles pour :

- Séparer des sections thématiquement différentes
- Marquer une transition importante dans le document
- Créer une pause visuelle dans un long texte

---

### Exemple de séparation

Contenu de la première section.

---

Contenu de la deuxième section, thématiquement différent.

---

Contenu de la troisième section.

## Bonnes pratiques

- Laissez une ligne vide avant et après la règle horizontale
- N'en abusez pas - trop de séparateurs peuvent fragmenter le contenu
- Utilisez-les de manière cohérente dans tout votre document
- Préférez les titres aux règles horizontales quand c'est possible
