# Zensical

## Introduction

Zensical est un générateur de documentation statique qui combine une configuration simple en TOML, un thème moderne performant et des fonctionnalités avancées pour produire des docs claires et rapides. Il s’inscrit dans la lignée des outils comme MkDocs, avec une attention particulière portée à la navigation instantanée, aux palettes de couleurs, et à l’UX.

## Ce que Zensical apporte

- Simplicité de configuration via `zensical.toml`
- Navigation puissante (sections, index de section, breadcrumbs, toc)
- Thème moderne avec mode sombre/clair et palettes personnalisables
- Performances (prefetch, navigation instantanée, pruning)
- Ergonomie (boutons copier dans les blocs de code, annotations, tooltips)

## À qui ça s’adresse ?

- Équipes produit et tech qui documentent des APIs, guides d’usage, runbooks
- Formateurs qui veulent un site de cours lisible et maintenable
- Projets open‑source cherchant une doc élégante sans complexité
