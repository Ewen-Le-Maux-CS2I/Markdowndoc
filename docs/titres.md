# Les titres

Les titres en Markdown utilisent le symbole `#`. Le nombre de `#` détermine le niveau du titre.

## Syntaxe

```markdown
# H1 Header
## H2 Header
### H3 Header
#### H4 Header
##### H5 Header
###### H6 Header
```

## Rendu

# H1 Header
## H2 Header
### H3 Header
#### H4 Header
##### H5 Header
###### H6 Header

## Bonnes pratiques

- Utilisez un seul H1 par page (titre principal)
- Respectez la hiérarchie des titres (ne sautez pas de niveaux)
- Laissez une ligne vide avant et après un titre
